01프로젝트 개요 & 운영 프로세스

버니스(농구팀)의 3개월 주기 실력 향상 테스트 기록을 관리·시각화하는 사이트입니다. 데이터는 구글 스프레드시트 한 개에 회차별 탭으로 누적하고, 사이트는 그 시트를 참조합니다.

운영 흐름

구글 스프레드시트 (1개) ├─ 탭: 2025-05-16 ← 1차 (현재 데이터) ├─ 탭: 2025-08-16 ← 2차 (3개월 뒤, 동일 형식으로 탭만 추가) └─ 탭: ... ← 회차마다 탭 추가 │ ▼ 사이트가 시트를 자동으로 읽어 반영 기록 사이트

핵심 원칙: 사용자는 탭 추가만 하면 되고, 탭 이름을 코드에 하드코딩하지 않고 자동 탐색하여 새 회차가 자동 반영됩니다.

02기술 구조도

호스팅/백엔드는 Cloudflare, 프론트는 React + Vite + TypeScript로 확정.

Data · 사용자가 관리
Google Spreadsheet — 탭 1개 = 테스트 회차 1개
  • 모든 탭이 동일한 컬럼 형식
  • 공유 설정: 링크 있는 모든 사용자 — 뷰어
▼  Google Sheets API v4 (read-only) · spreadsheets.get(탭 자동 탐색) + values.batchGet(전 회차)
Backend · Cloudflare Pages Functions
functions/api/records.ts (Workers 런타임)
  • API 키를 Secret으로 보관 GOOGLE_API_KEY
  • 시간 1'15" → 75초 파싱, "면제" 예외 처리
  • 목표 대비 달성률(0~100%) 정규화, 회차별 랭킹 계산
  • Cache API로 캐시(약 10분) → 깔끔한 JSON 반환

03데이터 모델

와이어프레임의 컬럼을 그대로 표준 형식으로 사용합니다.

컬럼형식목표치정렬 방향
이름텍스트——
드리블 셔틀런1'15"1'17" 이내낮을수록 좋음 ↓
골밑슛(양쪽)x / 105개 이상높을수록 좋음 ↑
자유투x / 52개 이상높을수록 좋음 ↑
45도 패스캐치x / 7 (면제 가능)5개 이상높을수록 좋음 ↑

셔틀런만 "낮을수록 좋음"이라 종목별 랭킹에서 방향 처리가 필요합니다. 랭킹·추이는 종목별로 원값 그대로 다룹니다 (종합 점수·정규화 없음). 개인 프로필의 레이더 차트는 종목별 값을 축으로 표시.

04사이트 구조도

🏀 버니스 실력 기록 │ ├─ 🏠 홈 (대시보드) "이번 분기 한눈에" │ ├─ 최신 회차 요약 (참여인원 / 날짜) │ └─ 종목별 팀 목표 달성률 게이지 │ ├─ 🏆 랭킹 "누가 제일 잘해?" │ ├─ [종목별] 셔틀런 / 골밑슛 / 자유투 / 45도패스캐치 │ ├─ 회차 선택 (1차 / 2차 / ...) │ └─ 목표 달성 뱃지 │ ├─ 👤 개인 프로필 "내 성장 추이" │ ├─ 선수 선택 │ ├─ 레이더 차트 (4종목 스킬 프로필) │ ├─ 종목별 추이 라인차트 (회차 누적) │ ├─ 개인 최고기록(PB) & 목표 대비 달성률 │ └─ 전 회차 대비 변화 (▲▼) │ ├─ 📈 전체 추이 "모든 선수 · 모든 회차 한눈에" │ ├─ 전체 인원 회차별 추이 (멀티라인 차트 / 매트릭스) │ └─ 종목별 보기 전환 │ └─ 🔒 관리자 · 시트 관리하기 "운영자 전용 · 관리자 코드" ├─ 기록지 만들기 — 오늘(KST) 회차 탭 생성 (참가자 선택: 활동만·가나다·기본 해제) └─ 기존 기록지에 사람 추가 — 특정 회차에 활동 선수 추가(맨 아래)

🔒 관리자 페이지의 두 기능은 시트에 쓰기를 하며, 확정 시마다 관리자 코드로 검증. 자세한 스펙은 아래 06 참고. (구현은 사이트 구축 단계에서)

05관리자 페이지 "시트 관리하기" — 기능 스펙

사이트 → 시트 쓰기 기능. 관리자 코드로 보호(생성/추가 확정 시마다 입력·서버 검증), 성공 시 캐시 자동 무효화.

① 기록지 만들기

항목동작
날짜오늘 (Asia/Seoul), YYYY-MM-DD. 오늘 탭 있으면 생성 안 함 + 안내
참가자 선택활동 선수만, 가나다 오름차순, 기본 전원 해제 → 체크한 사람만 탭에 생성
생성 방식최근 탭 복제 → 이름열을 선택 참가자 참조(가나다)로 재작성 → 점수 비움
원자성한 번의 batchUpdate — 실패 시 전체 롤백(반쪽 탭 없음)
이후"시트 열기" 링크 표시. 점수는 시트에서 직접 입력

② 기존 기록지에 사람 추가하기

대상 사람그 탭에 없는 활동 선수 멀티선택
위치맨 아래에 추가 (이름 참조 + 빈 점수)

06확정된 선택 & 남은 결정

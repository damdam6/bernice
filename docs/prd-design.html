02디자인이 확정한 기능 변경

#변경내용 · 근거
1전체 추이 페이지 삭제 → 개인 프로필로 통합하단 네비는 홈·랭킹·개인 3탭. 전체 비교는 개인 프로필의 종목 추이 차트에 전체 선수 배경 라인 + 본인 하이라이트 + 목표선으로 흡수. 인터뷰 확정 (2026-07-17)
2열람 게이트: 팀 패스코드 유지디자인에는 게이트 화면이 없지만(잠금 아이콘 = 관리자 로그인만), 시트에 실명 명단·기록이 있어 열람용 팀 패스코드는 유지로 확정. 게이트 화면 1개는 디자인에 추가돼야 함(§09 열린 항목). P2 인증 구조(팀/관리자 구분)는 그대로. 인터뷰 확정 (2026-07-17)
3랭킹 표시 정책 — 미측정·면제도 하단 노출순위권(기록 보유자) 아래에 미측정·면제 선수도 행으로 노출: 순위 자리에 —/면제, 뱃지로 구분. API 계약 변경 없음 — rankings(제외 유지) + sessions[].entries를 프론트에서 조합해 표시
4홈 요약 지표 추가 — 평균 목표 달성 %최신 회차 카드에 참여 인원 + 종목별 달성률의 평균(%) 병기. 프론트 파생값 — 계약 변경 없음
5관리자 IA — 기록 입력이 제1 기능"시트 관리" 홈 버튼 순서: 기록 입력(강조) → 참가자 추가 → 기록지 만들기. 기록 입력 동선에 날짜 선택 화면(회차별 완료 n/N)이 명시적 단계로 추가
6면제 토글은 면제 가능 종목에만입력 화면의 면제 토글은 면제 가능 종목(현재 45도 패스캐치)에만 노출. 서버·파서는 기존대로 모든 종목의 면제를 수용(시트 직접 편집 escape hatch와 의미론 동일) — UI 노출만 제한
7테마 교체 — 오렌지 → 인디고 + PretendardP0의 "농구 오렌지 테마"를 폐기하고 인디고(#3F51B5) 라이트 테마 + Pretendard로 전환 (§03)
